## Build do projeto
```bash
./mvnw clean install
```

## Rodando a aplicação
1. Rodando a aplicação usando Maven:
```bash
./mvnw spring-boot:run
```

`utilizando uma SERVER_PORT altearnativa`
```bash
SERVER_PORT=8082 ./mvnw spring-boot:run
```
2. Acessando o projeto:
-  Default URL: http://localhost:8080
